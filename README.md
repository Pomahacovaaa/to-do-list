## how was it made?
* html/js/css
* mixkit (sound effects, background music)
* some parts of W3Schools tutorials

## features
* light/dark mode
* basic to do list functions
* a confetti animation upon finishing a task (<a href="https://confetti.js.org/#usage">link to the code</a>)
* background sounds (birds, rain, song)
* a button to show you the exact time and date
